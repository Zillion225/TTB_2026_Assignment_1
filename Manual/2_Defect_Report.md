# Defect Report: Login Failure

**Issue Type:** Bug

> **Summary:** Users are unable to log in with valid credentials. This critical issue blocks all access to the system.

---

## Environment

- **Environment:** UAT

---

## Steps to Reproduce

1.  Open the application.
2.  Navigate to the **Login** page.
3.  Enter a valid **username**.
4.  Enter the correct **password**.
5.  Click the **Login** button.

---

## Expected vs. Actual Result

-   **Expected Result:** The user should be successfully authenticated and redirected to the home page.
-   **Actual Result:** The application fails to log the user in, despite correct credentials.

---

## Analysis

-   **Priority:** `High`
-   **Severity:** `Critical`
-   **Frequency:** Always

---

## Impact

This defect results in a complete blockage for users attempting to access the system.
